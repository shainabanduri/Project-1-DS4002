# Model Citizens

## Contents of the Repository
At the top level of the repository, the following files and folders are included:

- **README.md**  
  This file. It serves as an orientation guide for the repository, explaining the software requirements, folder structure, and step-by-step instructions for reproducing the results.

- **LICENSE.md**  
  Contains the MIT License, which specifies the terms under which the code and materials in this repository may be used and shared.

- **SCRIPTS/**  
  This folder contains all scripts used to clean data, run analyses, and generate results. Scripts should be executed in the order described in the reproduction instructions.

- **DATA/**  
  This folder contains the data used in the project. Raw data and any processed datasets required for analysis are stored here. The contents of this folder are necessary to reproduce the results.

- **OUTPUT/**  
  This folder contains all outputs generated by running the scripts, such as tables, figures, or other result files. These outputs allow users to verify that they have successfully reproduced the study results.
